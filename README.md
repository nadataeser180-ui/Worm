# 🔍 محلل الروابط الشامل - Worm

## 📌 نبذة عن المشروع

تطبيق ويب متقدم وشامل لجمع **كل المعلومات والبيانات** عن أي رابط على الإنترنت من خلال إدخال الرابط فقط!

يقوم بتحليل شامل للموقع ويستخرج:
- 🔗 معلومات الرابط والنطاق
- ⚙️ التكنولوجيات والأطر المستخدمة
- 📊 خدمات التحليل والإحصائيات
- 🔒 معلومات الأمان والشهادات
- 📧 عناوين البريد والهواتف
- 📱 روابط وسائل التواصل الاجتماعي
- 🖼️ الصور والوسائط
- 📝 البيانات الوصفية (Meta Tags)
- 🌐 جميع الروابط الداخلية والخارجية
- 🔬 بيانات متقدمة (JSON للمطورين)

## 🚀 طريقة الاستخدام

### الطريقة 1: استخدام محلل الروابط الشامل
```
1. افتح ملف analyzer.html في متصفحك
2. أدخل رابط الموقع في حقل الإدخال
3. اضغط على زر "تحليل الآن"
4. انتظر نتائج التحليل الشامل
5. استعرض البيانات من خلال التبويبات المختلفة
6. صدّر النتائج (JSON, PDF, أو نسخ)
```

### الطريقة 2: استخدام محرك البحث البسيط
```
1. افتح ملف index.html
2. ابحث عن الموارد والصور
3. استخدم الفلاتر للتصفية
```

## 📦 الملفات الأساسية وشرح وظائفها

### 1. `analyzer.html` 🎯
الصفحة الرئيسية للمشروع - تحتوي على:
- حقل إدخال الرابط
- تبويبات عرض النتائج
- أزرار التصدير

### 2. `comprehensive-analyzer.js` 🔬
المحرك الرئيسي للتحليل الشامل:
- جلب معلومات الصفحة
- استخراج البيانات الوصفية
- كشف التكنولوجيات
- استخراج جهات الاتصال

### 3. `ui-handler.js` 🎨
معالج الواجهة الرسومية:
- إدارة التبويبات
- عرض النتائج
- معالجة الأحداث

### 4. `url-parser.js` 🔗
معالج الروابط:
- تحليل الرابط إلى مكونات
- استخراج النطاق والنطاق الفرعي
- فحص صحة الرابط

### 5. `config.js` ⚙️
الإعدادات الأساسية:
- قائمة التطبيقات المعروفة
- الألوان والرموز
- الرسائل والنصوص

### 6. `style-analyzer.css` 🎨
التنسيقات المتقدمة:
- تصميم متجاوب
- رسوم متحركة سلسة
- نمط احترافي
