# مريم - محرك البحث 🔍

تطبيق ويب حديث للبحث عن الموارد والصور بواجهة عربية سهلة الاستخدام.

## المميزات ✨

- 🔍 **بحث ديناميكي** عن الموارد والصور
- 📱 **تصميم متجاوب** يعمل على جميع الأجهزة
- 🎨 **واجهة جميلة** بتصميم حديث
- 🌍 **دعم اللغة العربية** بالكامل
- ⚡ **أداء سريع** وتجربة مستخدم ممتازة

## المتطلبات

لا توجد متطلبات خاصة. التطبيق يعمل على أي متصفح حديث.

## كيفية الاستخدام

1. افتح ملف `index.html` في متصفحك
2. أدخل كلمة البحث في حقل البحث
3. اختر الفلتر (الكل، صور، موارد)
4. اضغط على زر البحث
5. شاهد النتائج
